╔══════════════════════════════════════════════════════╗
║     📸  AngLeeeee 摄影作品集 —— 使用说明                ║
║     97 张照片 + 1 部视频，6 个分类                       ║
╚══════════════════════════════════════════════════════╝

┌─────────────────────────────────────────────────────┐
│  📂 文件说明                                          │
│                                                       │
│  index.html       ← 主页面（不用动它）                 │
│  config.js        ← 改文字内容（名字、邮箱、简介等）    │
│  photos-data.js   ← 照片列表（自动生成）               │
│  generate-photos.js ← 重新扫描照片用                   │
│  photos/          ← 你的照片和视频                     │
│    ├── 风光/  (21张)                                  │
│    ├── 人物/  (36张)                                  │
│    ├── 街拍/  (15张)                                  │
│    ├── 体育/  (8张)                                   │
│    ├── 宠物/  (6张)                                   │
│    ├── 手机/  (11张)                                  │
│    └── 视频作品集.mp4                                  │
└─────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────┐
│  🎯 怎么改文字？                                       │
│                                                       │
│  打开 config.js，改等号右边 '' 里面的内容即可。         │
│  每个设置都有中文说明。                                 │
│  改完保存 → 刷新浏览器 → 生效！                        │
└─────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────┐
│  🌟 新功能                                            │
│                                                       │
│  1. 点击分类标题（如"风光"）→ 进入平铺浏览页面          │
│     像 Netflix 一样一张张浏览该分类的所有照片           │
│                                                       │
│  2. 每个分类行两侧有左右箭头 → 点击可以滑动浏览          │
│     也可以直接拖滚动条                                  │
│                                                       │
│  3. 视频展示区 → 直接播放你的视频作品                   │
│                                                       │
│  4. 点击服务项目 → 弹出你的联系方式                     │
└─────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────┐
│  📸 增加/删除照片                                     │
│                                                       │
│  1. 把新照片放进 photos/ 文件夹对应的分类里             │
│  2. 运行：node generate-photos.js                      │
│  3. 刷新浏览器                                        │
│                                                       │
│  给照片改个好听的名字：                                │
│  打开 photos-data.js，找到"标题"，改成你想要的名字      │
└─────────────────────────────────────────────────────┘
